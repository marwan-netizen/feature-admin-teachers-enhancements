# Glossary

### AI Engine
The internal module responsible for communicating with external AI providers and managing AI-related business logic.

### Adapter
A concrete implementation of an interface that handles communication with an external system (e.g., `GroqProvider`).

### Clean Architecture
An architectural pattern that focuses on the separation of concerns and the isolation of business logic from external frameworks.

### DTO (Data Transfer Object)
A simple object used to pass data between different layers of the application without carrying any business logic.

### Jitsi
An open-source multiplatform set of voice, videoconferencing, and instant messaging applications used here for live classroom sessions.

### MCQ (Multiple Choice Question)
A form of assessment in which respondents are asked to select the best possible answer out of the choices from a list.

### Port (Interface)
An abstract definition of a capability that the application needs, implemented by an adapter.

### Soft Delete
A method of deleting data where a record is marked as deleted but remains in the database for audit or recovery purposes.

### TTS (Text-to-Speech)
The artificial production of human speech by a computer system.
