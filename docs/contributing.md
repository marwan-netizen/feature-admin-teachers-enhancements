# Contributing to LingoPulse AI

First off, thank you for considering contributing to LingoPulse AI!

## 🤝 Code of Conduct

Help us keep this project open and inclusive. Please be respectful to other contributors.

## 🛠 How to Contribute

### 1. Reporting Bugs
-   Search existing issues to see if the bug has already been reported.
-   If not, open a new issue with a clear title, description, and steps to reproduce.

### 2. Suggesting Enhancements
-   Open an issue describing the feature and why it would be beneficial.
-   Wait for feedback before starting implementation.

### 3. Pull Request Process
1.  Fork the repository and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  If you've changed APIs, update the documentation.
4.  Ensure the test suite passes (`pytest`).
5.  Open a Pull Request with a clear description of your changes.
6.  The PR must be reviewed and approved by at least one maintainer before merging.

## 📝 Documentation

Documentation is just as important as code. If you find a typo or a missing guide, please submit a PR!
- All documentation is located in the `docs/` directory.
- Use Markdown for all documentation files.
