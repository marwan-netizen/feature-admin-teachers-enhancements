# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-05-20
### Added
- Initial release of the LingoPulse AI platform.
- Modular Clean Architecture implementation.
- AI-powered test generation for Reading, Listening, Writing, and Speaking.
- Evaluation engine using Google Gemini.
- Virtual classroom system with assignment management.
- Jitsi Meet integration for live sessions.
- Comprehensive documentation system.

### Changed
- Migrated from legacy Laravel codebase to Django 5.x.
- Refactored AI integration to use Port/Adapter pattern.

### Fixed
- Fixed session management issues during multi-stage tests.
- Improved error handling for AI provider timeouts.
